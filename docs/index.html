多 Agent 流程

每个 Agent 都有独立任务和输出，最终汇总成 GitHub README、结构化 evidence 和可截图页面。

01

Planner

定义目标用户、核心痛点、成功指标和 Token 使用理由。

02

Architect

设计模块边界、数据流和可复现的运行约束。

03

Builder

生成工程计划、命令入口和证明包交付物。

04

Reviewer

检查夸大陈述、复现风险和申请材料缺口。

05

Curator

整理申请表可用叙述、截图点和结构化证据。

复现命令

默认 mock 模式不用 API key，方便评审先跑通；有 MiMo key 后可切换到 live 模式。

git clone https://github.com/666-LGTM/imo-orbit-agent-kit.git
cd imo-orbit-agent-kit
python3 -m venv .venv
source .venv/bin/activate
python -m pip install -e .
python -m orbit_agent_kit.cli run --idea "生成 MiMo Orbit 申请证明包" --out reports/demo
